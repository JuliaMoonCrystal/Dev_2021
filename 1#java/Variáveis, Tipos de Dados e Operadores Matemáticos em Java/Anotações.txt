O que é uma variavel
-é um espaço na memoria do computador, onde se pode guardar valores
-------------------------
Instancia:objeto
Classe:classe
Local: dentro de métodos
Parametros: na assinatura do metodo

padrão Java-fortemente tipada

<Visibilidade> <modificador> nomeVariavel= <valorInicial>

-Não começar com números
-Evitar caracteres especiais
-São case-sensitive
-Sem espaço
-Não pode usar palavra reservada

-Boas práticas
-Sempre começar com letra minuscula
-nomes expressivos
-nome camelo
-nomes compostos separados com _ e inicia com letra maiuscula

Tipos das variaveis
-São valores e consequentemente operações que as váriaveis podem assumir e sofrer,respsctivamente.

-Estatica(forte) x Dinâmica(fraca)
-Promitivo vs Composto

Textual: String,char
Numerico:int ,double,byte,short,long,float
Lógico: boolean
Objeto:

Operadores lógicos
-São simbolo especiais quais são capazes de realizar ações especificas em um,dois ou três operadores.
-pós-fixado: y++ ou y--
-prefixado: ++x, --x
-aritmético:+,-,*,/,%
-atribuição:=,++,-=,*=,/=

Conceituação
-é a trasnformação de uma determinada variável de tipos menos especificos para um tipo mais especifico.
Upcast(implicito)
-Promoção de um tipo de dado com menor capacidade de armazenamento, para um dado maior

DownCast(expicito)
-Promoção de um tipo de dado com maior capacidade de armazenamento, para um dado menor
